# 🛡️ DesokupaAlicante — Сайт для компании по выселению сквоттеров

**Проект:** Одностраничный сайт для компании desokupación в Аликанте (Испания)

## Статус: 🟢 MVP готов

## Структура проекта

```
ocupas/
├── README.md              ← Этот файл
├── competitor-analysis.md ← Анализ конкурентов
├── site/                  ← Сайт (готов к деплою)
│   ├── index.html         ← Главная страница (исп+рус)
│   ├── css/
│   │   └── styles.css     ← Стили
│   ├── js/
│   │   ├── i18n.js        ← Переводы (es/ru)
│   │   └── main.js        ← Интерактивность
│   └── images/            ← Сюда — фото/лого
```

## Технологии

- **Чистый HTML/CSS/JS** — без фреймворков, максимальная скорость
- **Font Awesome** — иконки (через CDN)
- **Google Fonts (Inter)** — шрифт
- **Двуязычность** — испанский (основной) + русский (переключатель)

## Деплой

Варианты (самые простые):
- **Netlify** — просто перетащить папку `site/`
- **Vercel** — то же самое
- **GitHub Pages** — залить в репозиторий
- Любой хостинг (Apache, Nginx) — просто скопировать файлы
